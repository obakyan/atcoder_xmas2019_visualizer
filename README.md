This is a visualizer tool for https://atcoder.jp/contests/xmascon19/tasks/xmascon19_a .

# Usage
Copy resources ( `design.png` and `pieces` folder) to bin folder.

```
bin[folder]
  - xmas2019_a_visualizer.exe
  - design.png
  - pieces[folder]
```

Click tiles to replace them.
